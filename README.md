## 💻 Pré-requisitos

Antes de começar, verifique se você atendeu aos seguintes requisitos:

- Você instalou a versão mais recente de `< Node / NPM / PostgreSQL>`
- Você tem uma máquina `<Windows / Linux / Mac>`.
- Você tem um clone do banco de dados

## ☕ Usando <IANES>

Para usar <IANES>, siga estas etapas:

Dentro do diretório Front-end:
```
npm install
npm run dev
```

Dentro do diretório Back-end:
```
npm install
node app.js
```
